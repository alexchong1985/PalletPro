# PalletPro

PalletPro is a free, fast, and completely client-side pallet loading calculator. It helps users find the optimal arrangement of boxes on a pallet to maximize space utilization.

## Features

- **Fast & Lightweight**: Sub-second load times with pure HTML, CSS, and JavaScript.
- **3D Visualization**: Interactive 3D view of the loaded pallet using Three.js.
- **Optimal Packing Algorithm**: Tests multiple orientations and patterns (Grid, Brick) to find the best fit.
- **Layer-by-Layer View**: Breakdown the loading pattern layer by layer.
- **Constraints**: Support for maximum pallet height and weight.
- **Mobile Friendly**: Responsive design for use on warehouses and on the go.

## How it Works

1. Enter your Pallet dimensions (Length, Width, Max Height).
2. Enter your Box dimensions (Length, Width, Height) and Weight.
3. Set any constraints like Maximum Pallet Weight.
4. Click **Calculate Packing** to see the results and 3D visualization.

## Technologies Used

- **Three.js**: For 3D rendering.
- **JavaScript (ES6+)**: Core logic and algorithm.
- **CSS Grid/Flexbox**: Responsive UI.
